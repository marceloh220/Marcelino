  Module Math has mathematics functions.
  
  This module provides...
  
   Trigonometrics functions:
   sin(x)						Sine math function
   cos(x)						Cosine math function
   tan(x)						Tangent math function
   arcsin(x)					Inverse Sine math function
   arccos(x)					Inverse cosine math function
   arctan(x)					Inverse tangent math function
   
   Exponential function:
   pow(base, exponent)			Exponential math function
   euler(exponent)				Exponential function with euler number base
   
   Square functions:
   sqrt(base)					Square root math function
   sqrt(base, radical)			Square math function with base and exponential
   
   Logarithms functions:
   log(x)						Logarithm math function with base 10
   log(base, x)					Logarithm math function
   ln(x)						Natural logarithm math function
   
   Utils functions:
   abs(x)						Absolute math function
   max(x, y)					Returns the largest number between two passed numbers
   min(x, y)					Returns the less number between to passed numbers
   trunc(x)						Trunc the number
   round(x)						Round the number
   random(big)					Make a random number between 0 at the big number
   random(small, big)			Make a random number between a small number at the big number
   
   Convertion functions:
   rad(deg)						Convert degress to rad
   deg(rad)						Convert rad to degress
   inch(cm)						Convert centimeter to inch
   cm(inch)						Convert inch to centimeter
   tobcd(dec)					Convert decimal to BCD
   todec(bcd)					Convert BCD to decimal
   omega(freq)					Convert frequency to rad per seconds
   map(x, in_min, in_max, out_min, out_max)
   
   
   Constants functions:
   pi()							PI number with 6 precision decimal
   euler()						EULER number with 6 precision decimal
